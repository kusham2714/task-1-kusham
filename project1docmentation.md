Rule-Based Chatbot using Python
Introduction

This project is a simple Rule-Based Chatbot developed using Python. The chatbot interacts with users and provides responses based on predefined rules. It can answer basic questions such as greetings, asking its name, and responding to simple conversations. The main purpose of this project is to understand how chatbots work using conditional statements and loops in Python.

Objective

The objective of this project is to create a basic chatbot that can communicate with users through text-based conversations and respond according to predefined commands.
Tools and Technologies Used
Python
VS Code
Terminal
Working of the Project

The chatbot continuously takes input from the user and checks whether it matches any predefined condition. Depending on the input, it returns an appropriate response.

For example:

If the user types "hello" or "hi", the chatbot greets the user.
If the user asks "how are you", the chatbot replies with a predefined response.
If the user asks the chatbot's name, it introduces itself.
If the user types "bye" or "exit", the chatbot ends the conversation.
For any unknown input, the chatbot displays a default message.
Features
Interactive conversation with the user
Handles greetings and simple questions
Accepts input in any letter case using the lower() function
Exit option to stop the chatbot
User-friendly and easy to understand
