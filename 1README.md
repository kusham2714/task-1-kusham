# Rule-Based Chatbot Using Python

## Overview

This project is a simple Rule-Based Chatbot developed using Python. The chatbot interacts with users and provides responses based on predefined rules and conditions.

The project was created to understand the basic concepts of chatbot development, user input handling, loops, and conditional statements in Python.

## Features

* Responds to greetings such as "hello" and "hi"
* Answers simple questions
* Introduces itself when asked
* Handles unknown inputs
* Allows users to exit the conversation

## Technologies Used

* Python

## How It Works

The chatbot continuously accepts user input and compares it with predefined conditions using if-elif-else statements.

Supported Commands:

* hello / hi
* how are you
* what is your name
* bye / exit

If the user enters an unknown query, the chatbot displays a default response.

## Sample Output

You: hello

Bot: Hello! How can I help you?

You: what is your name

Bot: I am a Rule-Based Chatbot.

You: bye

Bot: Goodbye! Have a nice day.
